## Why

Criar um sistema de gerenciamento de bookmarks compartilhado para grupos privados, inspirado no linkding, com foco em performance, minimalismo e busca eficiente.

## What Changes

- Sistema de bookmarks com CRUD completo
- Sistema de tags com busca filtrada
- Busca textual livre em todos os campos
- Suporte a bundles de links
- Tema minimalista com cor predominante verde bélico
- Armazenamento otimizado para páginas estáticas (Astro)

## Capabilities

### New Capabilities
- `bookmark-system`: Sistema completo de gerenciamento de bookmarks
- `tag-search`: Sistema de busca por tags com filtragem múltipla
- `full-text-search`: Busca textual livre em todos os campos
- `link-bundles`: Sistema de agrupamento de links em bundles
- `minimal-theme`: Tema minimalista com cores verde bélico
- `static-storage`: Armazenamento otimizado para páginas estáticas

### Modified Capabilities

## Impact

- Frontend em Astro com tema minimalista
- Backend opcional (PostgreSQL) para funcionalidades avançadas
- Sistema de busca otimizado para performance
- Integração com favicon e ogthumb para links
- Sistema de autenticação para grupos privados
