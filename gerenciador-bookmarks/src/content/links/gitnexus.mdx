GitNexus é um motor de inteligência de código "Zero-Server" que indexa repositórios em um grafo de conhecimento para fornecer contexto arquitetural profundo para agentes de IA. Ele utiliza Tree-sitter para mapear dependências, cadeias de chamadas e clusters funcionais, permitindo análise de impacto, rastreamento de fluxo de execução e renomeação de múltiplos arquivos via Model Context Protocol (MCP).
