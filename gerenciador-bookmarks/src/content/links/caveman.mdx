Skill para Claude Code (e plugin para Codex) que comprime a verbosidade do agente trocando frases completas por "grunts" cavernícolas. Benchmarks com tiktoken mostram redução média de 61% nos tokens de saída em tarefas como busca, edição de código e Q&A.
